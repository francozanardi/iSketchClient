Más información

iSketchClient fue desarrollado con el fin de intentar solucionar los inconvenientes a la hora de jugar iSketch. La idea principal es reducir la complejidad al usuario para poder acceder al juego.

Para poder instalar iSketchClient requiere tener WinRAR instalado en su equipo, dado que el instalador está comprimido en un archivo RAR. Esta compresión se realiza para evitar bloqueos de Microsoft Defender SmartScreen en el instalador por editor desconocido (es decir, el software no fue desarrollado por una empresa que posea certificados).

Si usted no tiene instalado WinRAR, puede intentar descargar e instalar directamente el archivo ejecutable desde aquí.

Puede ver todo el código fuente en Github

Si tiene alguna duda, puede contactarme vía mi email personal francozanardi97@gmail.com.

¿No funciona?

Si ya descargó iSketchClient y la página web lo detecta pero aún así no puede acceder al juego, puede que se deba a alguna de las siguientes causas:

  • Usted tiene una tarjeta gráfica Nvidia, lo que puede ocasionar ciertos problemas de compatibilidad. Puede ver más sobre eso aquí, en la sección The Shockwave 3D Bad Driver List Bug.
  • iSketchClient requiere que su sistema operativo tenga instalado Microsoft .NET Framework 4.
