More info

iSketchClient was developed with the aim of trying to solve problems to play iSketch. The main idea is to reduce the complexity to the user to be able to access the game.

In order to install iSketchClient you need to have WinRAR installed on your computer, since the installer is compressed into a RAR file. This compression is done to avoid Microsoft Defender SmartScreen blocks in the installer due to unknown publisher.

If you do not have WinRAR installed, you can directly try to download and install the executable file from here.

You can see all the source code in Github

If you have any questions you can contact me via my personal email francozanardi97@gmail.com.

Does not work?

If you have already downloaded iSketchClient and the web page detects it but you still cannot access the game, it may be due to one of the following causes:

  • You have an Nvidia graphics card, which can cause some compatibility issues. You can see more about that here, in The Shockwave 3D Bad Driver List Bug section.
  • iSketchClient requires that your operating system have Microsoft .NET Framework 4 installed.
